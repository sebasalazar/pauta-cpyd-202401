\doxysubsection{Descripción detallada}
Clase para representar un año y un mes. 

La clase \doxylink{classYearMonth}{Year\+Month} proporciona una representación y manipulación de una combinación de año y mes. Esta es una idea robada de Java 8+, que es útil para trabajar con año y mes de una forma resuminda.
